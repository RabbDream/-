
## 分发糖果(leetcode-135 2020/12/24)
老师想给孩子们分发糖果，有 N 个孩子站成了一条直线，老师会根据每个孩子的表现，预先给他们评分。

你需要按照以下要求，帮助老师给这些孩子分发糖果：

每个孩子至少分配到 1 个糖果。
相邻的孩子中，评分高的孩子必须获得更多的糖果。
那么这样下来，老师至少需要准备多少颗糖果呢？

```h

示例 1:
输入: [1,0,2]
输出: 5
解释: 你可以分别给这三个孩子分发 2、1、2 颗糖果。

示例 2:
输入: [1,2,2]
输出: 4
解释: 你可以分别给这三个孩子分发 1、2、1 颗糖果。
     第三个孩子只得到 1 颗糖果，这已满足上述两个条件。

```

```h
思路: 至少需要准备多少颗糖果,于是想到使用贪心去计算,根据题意,两次遍历,一次从左往右遍历,确定符合条件所需的最小糖果,第二次从右往左遍历,计算当前位置符合条件的最小糖果数,和第一次遍历时当前位置的最小糖果数比较,最大值则为当前位置的最终结果

/**
 * @param {number[]} ratings
 * @return {number}
 */
var candy = function(ratings) {
  // 至少需要准备多少颗糖果
  let res = 0
  // 孩子的个数
  let len = ratings.length
  // 初始化第一次从左往右遍历时每个孩子需要的最小糖果数组
  let left = new Array(len).fill(0)
  // 第一个孩子至少为1个糖果
  left[0] = 1
  // 遍历求出其他孩子所需的最小糖果
  for(let i=1;i<len;i++){
    // 若当前孩子的得分比前一个孩子高,贪心将该孩子的糖果数比前一个孩子多一个,此时总数最小
    if(ratings[i]>ratings[i-1]){
      left[i] = left[i-1] + 1
    // 若当前孩子的得分不比前一个孩子高,给该孩子分配一颗糖果
    }else{
      left[i] = 1
    }
  }
  // 第二次从右往左遍历糖果,最后一个孩子初始化为1个糖果
  let right = 1
  // 最后一个孩子至少需要的最小糖果数
  res = Math.max(right,left[len-1])
  // 从右往左开始遍历
  for(let i=len-1;i>=1;i--){
    // 若前一个孩子比当前孩子分数高,则前一个孩子的糖果数比当前孩子多一个
    if(ratings[i-1]>ratings[i]){
      right++
    // 否则前一个孩子的糖果数为1个
    }else{
      right = 1
    }
    // 比较两次遍历中的最大值,即为满足条件的最小值
    res+=Math.max(right,left[i-1])
  }
  return res
};
```
